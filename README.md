# Stets Home — Приложение для управления умным домом

**Дипломный проект** по курсу «Системный аналитик» (Яндекс Практикум)

### О проекте
Приложение для управления умными домашними устройствами. Позволяет пользователю контролировать устройства и автоматически управлять режимом энергосбережения.

Проект выполнен по реальному ТЗ заказчика (компания Stets) на основе интервью с представителем бизнеса.

### Что входит в проект

**Артефакты системного анализа:**
- DFD-диаграммы (контекстная + 1 уровень)
- ER-диаграмма базы данных
- Диаграмма структуры интерфейса
- User Story Map (карта пользовательских историй)
- Словарь данных
- Программа и методика испытаний (ПМИ)

**Прототип интерфейса:**
- Интерактивный прототип приложения в Figma (ссылка ниже)
